## 🎯 Objetivos
#### Desenvolver uma aplicação em Java capaz de:
    - Consumir e/ou criar uma API em linguagem Java;
    - Coletar, organizar e apresentar dados de forma estruturada ao usuário;
    - Realizar persistência e consulta de dados em um banco de dados.
## 📦 Entregáveis
#### Os alunos deverão entregar:
    - Projeto Java completo;
    - Documento contendo:
    - Descritivo da solução desenvolvida;
    - Explicação do funcionamento da API;
    - Apresentação da solução desenvolvida (item obrigatório para avaliação do trabalho).
## 📊 Critérios de Avaliação
#### Serão considerados os seguintes aspectos:
    - Acesso a uma API externa (de livre escolha);
    - Consumo de dados da API escolhida e/ou criação de uma API própria (ambos é altamente desejável);
    - Implementação de endpoints;
    - Tratamento e organização dos dados recebidos da API, com foco na apresentação ao usuário;
    - Persistência dos dados em banco de dados;
    - Organização do projeto;
    - Separação de responsabilidades (arquitetura em camadas);
    - Fidelidade à especificação proposta;
    - Funcionamento correto do programa
